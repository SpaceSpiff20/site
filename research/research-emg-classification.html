EMG Gesture Classification with Multinomial Logistic Regression and a Shallow Neural Network

Researcher and Author Stanford, CA September 2025 – December 2025

Abstract:
This project investigates whether a relatively simple generalized linear model, multinomial logistic regression (softmax regression), can accurately classify hand and wrist gestures from low-cost surface electromyography (SEMG) using only four channels and lightweight time-domain features. The project targets six classes using signals acquired from Myoware EMG sensors. After explicitly trimming unstable gesture edges, softmax regression achieves about 92% test accuracy on real data, while a shallow MLP yields a modest improvement to around 93%. Most residual errors occur between fine finger gestures (point vs. pinch). These results suggest that classical linear models already perform strongly with carefully engineered features and segmentation, and small nonlinear models provide incremental but meaningful gains.

Key Focus Areas:

  • Softmax Regression Performance: To determine how far a well-designed EMG pipeline with multinomial logistic regression could go for multi-class gesture recognition, the model achieved around 92% test accuracy on trimmed real EMG with 300 ms windows, with most errors confined to fine finger gestures (point vs. pinch).
  • Shallow MLP Gain: Assessing whether modest nonlinearity provides additional gains, a shallow MLP improved test accuracy to around 93% on 200 ms windows, mainly by reducing confusions among overlapping multi-muscle gestures. The gain was only about 1-2 percentage points, indicating the feature space was already close to linearly separable.
  • Feature Engineering and Segmentation: The study used simple time-domain features (MAV, RMS, Waveform Length) with four SEMG channels. A key challenge was mitigated by explicitly trimming unstable gesture edges, where only the central 40% of each contiguous gesture block was marked as "stable".
